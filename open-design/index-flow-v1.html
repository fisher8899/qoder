1
年度指标设定
条件财务处管理员 部门绩效管理员 ▼
创建年度业绩指标设定考核组 管理员
条件财务处绩效考核管理员创建年度指标设定组，添加成员名单、业绩指标模版，设定开始/结束日期
操作内容
创建考核组 → 维护成员名单 → 绑定指标模版 → 设定时间窗口
系统字段
考核组名称、考核类型（业绩指标设定）、开始日期、结束日期、状态
关键规则
考核组名称默认格式：XXXX年XX月业绩指标设定组；需先配置考核对象（组织/人员）后才可启动
部门录入年度考核目标 部门操作
部门绩效管理员与部门负责人、分管领导线下沟通后，在系统中逐项录入年度考核指标（大类→小类→考核内容三级结构）
录入结构
大类（重点工作/基础工作/控制指标等）→ 小类 → 考核内容（含指标目标、权重、考核标准、考核部门）
支持方式
逐条录入 / 批量导入（按模版Excel）
指标分类
职能部门：重点工作(30) + 基础工作(40) + 控制指标(70)；分公司：经营指标(10) + 重点任务(20) + 控制指标(70)；公共：动态督办(50)、党建(60)、特殊贡献(80)、否决项(90)
三级审批流程 逐级审批
部门绩效管理员提交 → 部门负责人审批 → 分管领导审批 → 条件财务处管理员审批
审批流程
草稿 → 提交 → 待部门负责人审批 → 通过/退回 → 待分管领导审批 → 通过/退回 → 待财务处审批 → 通过/退回 → 审批通过
任何环节退回均退回到部门绩效管理员，修改后重新提交全流程
目标更新规则：年度指标原则上中途不调整。如确需变更，部门绩效管理员需线下联系条件财务处管理员执行退回，修改后重新走审批流程。
2
月度指标发布
条件财务处管理员 ▼
创建月度考核组 管理员
创建当月考核组，选择考核类型（月度考核），绑定考核模版，设定开始/结束日期
考核组配置
名称默认：2026年X月月度考核组；类型：月度考核；绑定考核对象
操作按钮
发布考核指标 → 启动考核 → 查看进度 → 启动他评 → 预发布 → 正式发布
发布月度考核指标 管理员
将年度指标按月拆分后发布，各部门可查看当月业绩考核指标
发布后效果
各部门绩效管理员可查看本部门当月考核指标明细，包括大类、小类、考核内容、指标目标、权重、考核标准、考核部门
启动月度考核 管理员
启动当月考核流程，各部门绩效管理员进入自评阶段
3
部门自评
部门绩效管理员 ▼
逐项填报自评 部门操作
部门绩效管理员收集当月指标完成情况，与部门负责人沟通确认后，在系统逐项录入自评反馈和自评得分
填报字段
实际完成情况（500字）、自评得分（0-120分）、证明附件（PDF/图片/压缩包，≤20M）
进度控制
完成进度 = 已打分指标数 / 总指标数；进度100%方可提交
自动计算
自评结果 = 自评得分 × 月度权重（保留2位小数）
提交自评 部门操作
确认无误后提交，状态更新为"已提交"。可随时保存草稿。
4
互评打分
条件财务处管理员 考核员/分管领导 ▼
启动他评打分 管理员
全部部门自评完成后，条件财务处管理员在考核组点击"启动他评"，开放部门互评通道
部门考核员对其他部门打分 考核员操作
各部门绩效考核员/分管领导对负责的被考核部门指标逐行打分，系统自动计算得分×权重
打分方式
按部门展示：逐部门完成打分；按指标展示：比较不同部门在同一指标的得分
打分规则
得分范围：0-120分；考核结果 = 得分 × 月度权重（保留2位小数）
提交条件
全部指标打分完成后方可提交；支持中途保存；页面显示总打分部门数、待打分指标数、已完成指标数、完成百分比
5
管理员复核与修正
条件财务处管理员 ▼
复核审阅全部考核数据 复核
全部部门互评提交完成后，条件财务处管理员逐部门逐指标复核，可对不完整数据进行补充或纠正
得分计算规则（核心）
优先级：先取条件财务处管理员打分 → 若为空则取考核部门打分 → 按大类汇总 → 按被考核部门汇总
系统记录管理员对每个部门每个指标的打分和修正说明
6
绩效预发布
条件财务处管理员 ▼
预发布考核结果 管理员
将月度考核结果预发布给部门绩效管理员和部门负责人查看，接受申诉。可"取消预发布"（可逆操作）
预发布后效果
部门绩效管理员和部门负责人可查看完整的评价打分信息（自评分、他评打分、管理员打分、最终得分）。如有异议，进入线下申诉流程。
7
申诉处理
条件财务处管理员 部门绩效管理员 ▼
线下申诉 → 系统创建申诉单 管理员
部门线下申诉成功后，条件财务处管理员在系统创建申诉单：选择申诉部门、打分部门、填写申诉说明、上传申诉材料
申诉单字段
申诉部门、打分部门（可多部门）、申诉说明、附件材料、状态（草稿/待重新评估/已重新打分）
退回操作
选择需重新打分的部门（可多选），将该部门的月度考核数据退回
打分部门重新评估 部门操作
被退回的打分部门在"评估反馈"列表看到状态为"待重新评估"，针对申诉指标重新打分并提交，数据直接覆盖原记录
管理员重新计算得分 管理员
条件财务处管理员重新复核业绩考核数据，重新计算最终月度业绩考核得分
申诉截止：在申诉截止日期后，不再接受新的申诉。管理员选择合适时间进行正式发布。
8
正式发布
条件财务处管理员 ▼
发布月度考核结果 管理员
条件财务处管理员正式发布当月业绩考核，全部数据冻结。可"取消发布"（可逆操作）
发布后权限
部门绩效管理员、部门负责人：可查询本次考核结果数据
条件财务处管理员：可导出明细数据表和汇总数据表
月度考核系数将与薪酬绩效奖金挂钩
本次月度考核完成。考核结果数据可用于薪酬绩效计算。下月重复步骤 2-8。
