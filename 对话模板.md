# 对话模板

当用户说“按我们的对话模板”时，按以下格式回复：

1. 全程使用中文。
2. 优先简洁直接，不写多余寒暄。
3. 每次回复都使用固定结构。
4. 如果没有补充内容，可以省略“【补充说明】”块。
5. 每次回复结尾都加同一条长虚线分隔，表示这一轮问答结束。

固定格式：

```text
【我的回答】
这里写直接回复内容

【补充说明】
有需要时再补充

----------------------------------------------------------------------
```

补充约定：

- 如果内容较短，只保留“【我的回答】”也可以。
- 不要求复述“【你的问题】”。
- 如果用户另行指定格式，以用户最新要求为准。
